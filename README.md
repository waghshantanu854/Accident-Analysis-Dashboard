# Accident-Analysis-Dashboard
Road Accident Analysis Dashboard (2021-2022)
This Power BI project provides actionable insights into road accident data for 2021 and 2022. The dashboard supports the Ministry of Transport, Police Force, and key stakeholders in enhancing road safety.

Focus: Total Casualties and Accidents, identifying high-risk areas, and tracking monthly trends.

1.Important Dashboard Insights
The dashboard addresses critical questions for safety improvement:

Core Performance: See Total Casualties and Total Accidents for the Current Year. Know your Year-over-Year (YoY) growth.

Severity Impact: Analyze Casualties by Accident Severity (Fatal, Serious, Slight). This pinpoints areas needing severe outcome reduction.

Vehicle Risk: Identify which vehicle types (e.g., Bus, Agricultural) contribute most to Current Year Casualties.

Time Trends: Review the Monthly Trend comparing Current Year vs. Previous Year casualties. Pinpoint high-risk months.

Road Environment: Understand casualty distribution by Road Type (e.g., Single Carriageway, Dual Carriageway).

Location Focus: Target specific Area/Location and Day/Night periods for Current Year casualty reduction.

Geographic Hotspots: Identify Total Casualties and Total Accidents by Location for precise resource deployment.

2. Technical Details
Tool: Microsoft Power BI

Data Period: 2021 and 2022
Purpose: Data-driven decision-making for road safety stakeholders.
